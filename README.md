# my-git-cheat-sheet
Simple git

## Clone
To clone a repo:
```git clone https://github.com/Ola-B/my-git-cheat-sheet.git```

## Add new file
To track a new file:
```git add <filename>```
```git add -a```

## Status
```git status```

## Commit changes
```git commit -m "commit message"```
To stage and commit at same time:
```git commit -a -m "commit message"```
